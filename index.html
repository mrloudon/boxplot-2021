Box Plot

175.203 - Introduction to Psychological Research
Use this tool to generate and save box plots.

Enter some data into the form below and click Plot. The tool will then generate a box plot using the data provided.

Each row of data corresponds to a single data set. You can have more than one data set by clicking Add Row as required. Clicking the Delete Row button will delete the last row from the table. Only one row (data set) is shown by default but you can have as many as you want.

Enter a title and a Y axis label for your plot in the text areas provided. You can also choose to add a comment which will appear below the plot if desired.

You can enter multiple outliers by separating the values by commas. When you are happy with your plot you can print or download it by clicking theicon at the top right corner of the plot.
